Understanding Diffusion Models

Generative AI Concepts

1. Forward Process (Adding Noise)

$$q(x_t | x_{t-1}) = \mathcal{N}(x_t; \sqrt{1 - \beta_t} x_{t-1}, \beta_t \mathbf{I})$$

2. Reverse Process (Denoising)

$$p_\theta(x_{t-1} | x_t) = \mathcal{N}(x_{t-1}; \mu_\theta(x_t, t), \Sigma_\theta(x_t, t))$$

3. Simplified Training Objective

$$\mathcal{L}_{simple} = \mathbb{E}_{t, x_0, \epsilon} \left[ \| \epsilon - \epsilon_\theta(x_t, t) \|^2 \right]$$

import torch
import torch.nn as nn

# 1. Forward Process (Adding Noise)
def add_noise(original_image, timestep, noise_schedule):
    # This acts like our static TV screen formula.
    # It calculates exactly how much noise to add for a given step.

    # Generate random pure noise
    noise = torch.randn_like(original_image)

    # Get the noise amount for the current timestep
    alpha = noise_schedule[timestep]

    # Blend the original image with the static noise
    noisy_image = (alpha ** 0.5) * original_image + ((1 - alpha) ** 0.5) * noise

    return noisy_image, noise

# 2. The AI Model (Predicting Noise)
class SimpleDiffusionAI(nn.Module):
    def __init__(self):
        super().__init__()
        # In reality, this is a complex U-Net.
        # Here we use a simple network for understanding.
        self.network = nn.Sequential(
            nn.Linear(28*28, 128),
            nn.ReLU(),
            nn.Linear(128, 28*28) # Output predicts the noise pattern
        )

    def forward(self, noisy_image, timestep):
        # The AI looks at the noisy image and guesses the noise
        return self.network(noisy_image)

# 3. Training the Model
def calculate_loss(model, original_image, timestep, schedule):
    # Step A: Mathematically ruin the image
    noisy_img, true_noise = add_noise(original_image, timestep, schedule)

    # Step B: Ask the AI to guess what noise was added
    predicted_noise = model(noisy_img, timestep)

    # Step C: Check how wrong the AI was (Mean Squared Error)
    # The goal is to make this difference as close to 0 as possible
    loss = nn.functional.mse_loss(predicted_noise, true_noise)

    return loss
